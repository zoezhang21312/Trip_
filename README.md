# VoyageSync - 旅行协作应用

一个极致简约的旅行协作应用，采用奶油白背景和深森林绿文字设计。

## 功能特性

- **左侧侧边栏**
  - 常用工具箱：管理常用链接
  - 历史记录：查看过往行程

- **主内容区域**
  - 垂直列表流展示行程卡片
  - 每个卡片包含：
    - 中英双语标题
    - 地点描述
    - Google Maps 图标按钮
    - 勾选状态

- **顶部输入框**
  - 支持粘贴小红书笔记

- **Packing Checklist**
  - 悬浮式打包清单组件
  - 可添加、勾选、删除清单项目

## 技术栈

- React 18
- Vite
- Tailwind CSS

## 安装和运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 设计风格

- 背景色：奶油白 (#FFFEF7)
- 文字色：深森林绿 (#1B4D3E)
- 设计理念：极致简约
